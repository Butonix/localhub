Description
-----------

Communikit is short for Community Toolkit. It is an experimental project aiming to provide secure, localized, private social media for small to medium sized communities.

Proposed features
-----------------

* Posts
* Markdown content
* Events/calendar
* Notifications
* Messages
* Localization support
* Community settings
* User management
